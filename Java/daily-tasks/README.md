## How to Use

1. **Every morning:** Open today's file and fill in your plan
2. **During the day:** Update as you complete tasks
3. **End of day:** Review what you accomplished

## File Naming Convention
**Format:** `Day-XX-YYYY-MM-DD.md`

- **Day-XX:** Sequential day number (01, 02, 03...)
- **YYYY-MM-DD:** Actual date (2025-10-29)

**Examples:**
- `Day-01-2025-10-29.md` - First day on Oct 29
- `Day-02-2025-10-30.md` - Second day on Oct 30
- `Day-15-2025-11-15.md` - Day 15 on Nov 15

**Benefits:**
- Easy to see which day came first
- Know the exact date of each day
- Chronological sorting in file explorer
